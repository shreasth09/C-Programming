# 🏦 Bank Management System (BKS)

A file-based banking system that allows users to create accounts, deposit, withdraw, and check balances.

This project simulates real banking operations using structured file storage and demonstrates practical use of C programming.

## 🚀 Features
- Create account  
- Deposit money  
- Withdraw money  
- Check balance  

## 🧠 Concepts Used
- Structures  
- File Handling  
- Data Persistence  
- Menu-driven Interface
