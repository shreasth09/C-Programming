# 🎓 Student Record Management System (SRMS)

This project is a file-based system to manage student records efficiently. It allows users to add, view, search, and delete student data stored in files.

The system simulates a real-world database using file handling and demonstrates structured data storage and retrieval in C.

## 🚀 Features
- Add student record  
- View all records  
- Search student by ID  
- Delete student record  

## 🧠 Concepts Used
- File Handling  
- Structures  
- Searching & Filtering  
- CLI-based Menu System
