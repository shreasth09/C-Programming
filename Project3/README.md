# 📊 Log Analyzer (C)

A tool that processes log files and extracts useful information such as error logs, warnings, and statistics.

This project demonstrates real-world log analysis similar to backend systems.

## 🚀 Features
- Count ERROR/WARNING  
- Extract specific logs  
- Generate summary  

## 🧠 Concepts Used
- File Processing  
- String Matching  
- Data Analysis
